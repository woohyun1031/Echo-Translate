# Echo-Translate

크롬 기본 번역기를 쓸 때 언어가 통째로 한글로 덮어씌워져서, 원래 영단어가 뭐였는지 다시 확인하기 답답해서 만든 익스텐션. 
Echo-Translate는 이런 불편함을 없애기 위해, 원문을 지우지 않고 그 바로 아래에 `(번역된 한국어)`를 달아주는 심플한 크롬 확장 프로그램입니다.

## 어떻게 동작하나요?
비용이 드는 무거운 외부 번역 API를 쓰지 않고 크롬 내장 번역 기능을 그대로 활용합니다. 
번역기가 화면의 글자를 한글로 바꿔치기 하는 찰나의 순간을 캐치해서, 사라지는 원문을 낚아챈 다음 번역문 위에 다시 얹어주는 원리입니다. 무한루프나 메모리 누수가 없도록 설계해두어서 가볍게 켜두고 쓰기 좋습니다.

## 내 크롬에 설치해서 써보기
1. 크롬 주소창에 `chrome://extensions/`를 입력해서 확장 프로그램 페이지로 들어갑니다.
2. 우측 상단에 있는 **개발자 모드**를 켜주세요.
3. 좌측 상단에 있는 **압축해제된 확장 프로그램을 로드합니다** 버튼을 누릅니다.
4. 이 파일이 들어있는 `Echo-Translate` 폴더를 통째로 선택하면 설치 끝!
5. 이제 아무 영문 사이트나 들어가서 텍스트를 두고 우클릭 -> '한국어로 번역'을 실행해 보세요.
